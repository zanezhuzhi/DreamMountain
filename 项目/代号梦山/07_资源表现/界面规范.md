# 梦山 界面规范总则 / UI Spec Master

> **技术栈**: Unreal Engine 5 (UMG / Common UI)
> **风格基调**: 厚涂风格化 3D 下的 2D UI，温暖、通透、可亲近
> **核心依赖**: [资源表现总则](./00_资源表现总则.md)、[Vision 愿景](../00_项目核心/01_Vision_愿景.md)

---

## 1. 概述 / Overview

本规范定义梦山项目界面设计的原则、UE5 技术约束与厚涂风格的 UI 表现。具体界面（如主菜单、背包、协作提示等）需基于本文档创建独立界面规范。

---

## 2. 设计目标 / Design Goals

*   **沉浸优先**: 界面 **必须** 减少对攀爬与协作的打断；关键协作时刻 **禁止** 全屏弹窗。
*   **清晰可读**: 反馈 **必须** 明确；协作成功/失败、关系进度等需一眼可辨。
*   **情感一致**: UI 视觉 **必须** 与厚涂风格化 3D 世界协调，温暖、柔和，无硬边科技感。
*   **多端适配**: **必须** 支持触控（移动）与键鼠/手柄（PC）的双模操作。

---

## 3. UE5 技术规范 / UE5 Technical Specs

### 3.1 UI 框架

| 属性 | 定义 |
| :--- | :--- |
| **框架** | UMG (Unreal Motion Graphics) / Common UI |
| **布局** | Canvas Panel + 锚点自适应 |
| **分辨率基准** | 1920×1080 (16:9)；移动端支持竖屏 1080×1920 |
| **DPI 缩放** | 支持多档位；字体与图标需矢量或高分辨率以适配 4K |

### 3.2 输入与交互

| 平台 | 主要输入 | 备选 |
| :--- | :--- | :--- |
| 移动端 | 触控（点击、滑动） | - |
| PC | 键鼠、手柄 | 触控（触屏设备） |
| **必须支持** | 协作相关 UI 在攀爬过程中可单手/单手拇指触及 | - |

### 3.3 与 3D 场景的关系

*   **HUD / 浮层**: 攀爬中的血量、协作提示、搭档状态等 **必须** 轻量化，不遮挡视野。
*   **全屏界面**: 主菜单、背包、设置等可在安全时机展示；**禁止** 在关键协作节点强制弹出。
*   **世界空间 UI**: 协作节点、检查点等 **可以** 使用 3D 空间内 UI（Widget in World）增强沉浸。

## 4. 视觉风格 / Visual Style

### 4.1 厚涂风格在 UI 中的体现

| 元素 | 原则 | 说明 |
| :--- | :--- | :--- |
| **背景** | 半透明、柔和 | 避免纯色块；可使用渐变、轻微纹理、毛玻璃效果 |
| **边框/按钮** | 软边、有机形态 | 圆角、柔和描边；避免硬科技感 |
| **图标** | 手绘感、统一笔触 | 与厚涂 3D 角色/场景风格一致 |
| **字体** | 易读、温暖 | 推荐圆润、无衬线；避免冷硬科技字体 |

### 4.2 色彩规范

| 用途 | 原则 | 示例方向 |
| :--- | :--- | :--- |
| 主色 | 温暖、希望感 | 暖黄、琥珀、浅橙 |
| 辅色 | 自然、通透 | 浅蓝、青绿、云白 |
| 强调色 | 协作成功、正向反馈 | 柔和金/绿 |
| 警告/失败 | 柔和、非惊吓 | 浅红、暖灰，避免刺眼纯红 |

### 4.3 动效原则

*   **时长**: 界面进入/退出 200–400ms；微交互 100–200ms。
*   **缓动**: 使用 ease-out / ease-in-out；避免 linear 的机械感。
*   **反馈**: 点击、协作成功/失败 **必须** 有清晰的视觉与音效反馈。

---

## 5. 功能与场景原则 / Functional Principles

### 5.1 核心玩法相关 UI

*   **协作提示**: **必须** 在搭档进入可协作范围时给出明确提示；**必须** 在协作成功/失败时给予即时反馈。
*   **关系进度**: **应该** 在适当时机（如登顶、到达检查点）展示，不打断攀爬节奏。
*   **路线/地图**: **可以** 提供简洁的路线指引；**禁止** 过度 UI 化导致破坏沉浸。

### 5.2 社交与情境相关 UI

*   **挚友/搭档列表**: 入口清晰；**必须** 支持快速发起双人情境。
*   **陌生人互动**: 提示友善、非强制；保护社恐与独行玩家的体验。
