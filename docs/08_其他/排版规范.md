# 梦山 文档排版规范

> **适用范围**: 文档区（`docs/`）下所有 Markdown 文档。  
> **更新日期**: 2026-02-09

---

## 1. 专名与状态/动作的括号

- **统一使用直角引号「」**（U+300C / U+300D），用于：
  - 游戏内状态名：如「攀爬状态」「托举状态」
  - 动作/技能名：如「基础跳跃」「攀爬跳跃」「释放支点」
  - 专有名词或需要强调的术语：如「可攀爬斜面」「献祭周期」
- **不使用方头括号【】**（U+3010 / U+3011），避免与「」混用。

## 2. 书名与文档引用

- 文档名、规范名继续使用**书名号《》**，如《道具系统》《协作与救援》。

## 3. 其他

- 括号（）、英文/代码、表格等保持常规用法；新增或修订文档时请遵循上述括号约定。
