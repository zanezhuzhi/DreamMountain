### 1.1 引擎选型

| 属性 | 定义 |
| :--- | :--- |
| **引擎** | Unreal Engine 5 |
| **渲染管线** | [Default / Forward / Lumen — 待技术验证] |
| **目标平台** | iOS、Android、PC |
| **多端策略** | 移动端可降级材质与特效，保持风格一致 |

### 1.2 UE5 技术要点

*   **材质**: 支持自定义 NPR（非写实）材质；厚涂质感通过手绘贴图 + 风格化 Shader 实现。
*   **光照**: 温暖、通透；优先体积光、柔和阴影，避免硬边与过度 HDR。
*   **性能**: 移动端需控制 DrawCall、面数；Nanite/Lumen 使用需按目标平台评估。

## 2. 美术风格定义 (Art Direction)

### 2.1 核心风格：厚涂风格化 3D (Painterly Stylized 3D)

| 维度 | 定义 | 说明 |
| :--- | :--- | :--- |
| **质感** | 厚涂 / 笔触感 | 可见的笔触与色块，非写实光滑表面；类似油画/插画的体块感 |
| **造型** | 风格化 / 概括 | 轮廓概括、比例适度夸张；避免超写实或纯二次元 |
| **色彩** | 温暖、通透 | 饱和度适中，明度偏高；天空、远山、光照带希望感 |
| **光影** | 柔和、可亲近 | 柔和投影，避免强烈对比；与「旅途、互助、羁绊」情感基调一致 |

### 2.2 参考标杆 (Reference)

| 类型 | 参考作品 | 借鉴点 |
| :--- | :--- | :--- |
| **氛围** | 光遇 (Sky) | 柔和光照、通透场景、情感向氛围 |
| **手感与场景** | Peak | 攀岩场景、协作反馈、环境可信度 |
| **厚涂/风格化** | 原神、旷野之息等 | 手绘贴图、风格化材质、非写实光照 |

### 2.3 与 Vision 的衔接

*   **温暖、通透、可亲近** → 材质与光照 **必须** 传达柔软、安全、可探索感。
*   **自然/幻想山景** → 地形、植被、建筑 **必须** 兼顾自然感与适度幻想元素。
*   **避免阴暗、恐怖、暴力** → **禁止** 使用高对比、血腥、惊悚视觉元素。

## 4. 绝对边界 (Absolute Boundaries)

*   **[规则]**: 所有视觉与音频资源 **必须** 符合「温暖、通透、可亲近」的情感基调；不得出现恐怖、暴力、成人向内容。
*   **[规则]**: 厚涂风格化 3D **必须** 全项目统一；禁止混入写实、硬科幻或与世界观冲突的风格。
*   **[规则]**: 核心攀爬与协作的视觉反馈 **必须** 清晰可读，保障玩法可玩性优先于纯美术表现。
