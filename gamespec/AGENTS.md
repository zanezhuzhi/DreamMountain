# GameSpec 智能开发助手 / Intelligent Assistant

> **核心入口 / Entry Point**
> 本文件是 GameSpec 规范体系的唯一入口。

## 🚀 快速开始 / Quick Start

请严格按照以下顺序阅读并执行：

1.  **第一步：阅读核心法则**
    *   **[文档与目录规范](gamespec/01_规范标准/文档与目录规范.md)** (必读): 了解文件放在哪，怎么命名。这是**最高纲领**。
    *   **[游戏规范标准](gamespec/01_规范标准/游戏规范标准.md)** (必读): 了解文档怎么写，包含哪些章节。

2.  **第二步：选择身份 (或寻求导航)**
    *   **不知道该找谁？** 请直接加载 **[Spec 导航助手](gamespec/03_智能助手/00_Spec导航助手.md)**。它会帮您分析需求并分配任务。
    *   **已有明确目标？** 请直接加载对应的专家助手：
        *   **[首席制作人](gamespec/03_智能助手/首席制作人助手.md)**: 立项、创意、管理。
        *   **[技术总监](gamespec/03_智能助手/技术总监助手.md)**: 技术、性能、数据。
        *   **[系统策划](gamespec/03_智能助手/系统策划助手.md)**: 系统、数值、UI。
        *   **[核心玩法](gamespec/03_智能助手/核心玩法设计助手.md)**: 3C、战斗、手感。
        *   **[世界观](gamespec/03_智能助手/世界观设计助手.md)**: 历史、地理、设定。
        *   **[叙事](gamespec/03_智能助手/叙事设计助手.md)**: 剧情、台词、演出。

3.  **第三步：使用模板**
    *   **查重原则 (Check First)**: 在创建前，**必须**遍历目录。若发现同名或**语义近似**（如`Pet` vs `Follower`）的文档，**严禁新建**，必须读取原文件并进入[修改调优模式]。
    *   从 `gamespec/02_项目模板/` 复制对应的中文模板文件到你的项目目录中。

## 🛠️ 常用工具 / Tools

*   **验证工具**: 编写完文档后，运行 `python gamespec/04_自动化工具/lint_gamespec.py` 检查格式。
*   **技能库**: 查阅 `gamespec/06_技能合集/` 获取绘图、数值模拟等专项能力。
