## 交互流程 / Interaction Flow

### 流程: [流程名称] / Flow: [Flow Name]

```
[用户操作1] --> [界面响应1] --> [用户操作2] --> [结果]
```

**示例流程：出售物品**

```
点击物品 --> 显示物品详情 --> 点击出售 --> 显示确认弹窗 --> 确认 --> 出售成功提示
                                              |
                                              v
                                           取消 --> 关闭弹窗
```

| 步骤 / Step | 用户操作 / Action | 界面响应 / Response |
|-------------|-------------------|---------------------|
| 1 | 点击物品 | 显示物品详情面板 |
| 2 | 点击"出售"按钮 | 弹出确认对话框 |
| 3a | 点击"确认" | 扣除物品，增加金币，显示成功提示 |
| 3b | 点击"取消" | 关闭对话框，返回详情面板 |
