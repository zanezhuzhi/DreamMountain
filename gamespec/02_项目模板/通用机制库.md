# 通用机制库 / Common Mechanics Library

> **用途**: 定义全游戏共享的通用规则、状态效果和基础属性，避免重复定义。
> **引用方式**: 在其他文档中使用 `[机制:名称](../02_项目模板/通用机制库.md#anchor)` 进行引用。

---\n
## 1. 状态效果 / Status Effects

所有状态效果 **必须 (MUST)** 定义唯一的 ID 和持续时间规则。

### [ST_STUN] 眩晕 / Stun
- **ID**: `ST_STUN`
- **效果 / Effect**:
    - 角色 **无法移动 (MUST NOT)**。
    - 角色 **无法释放技能 (MUST NOT)**。
    - 播放眩晕特效 `fx_common_stun_loop`。
- **解除 / Remove**: 持续时间结束或被[净化]效果移除。

### [ST_INVINCIBLE] 无敌 / Invincible
- **ID**: `ST_INVINCIBLE`
- **效果 / Effect**:
    - 角色 **免疫所有伤害 (MUST)**。
    - 角色 **免疫所有控制效果 (MUST)**。
- **注意**: 无敌状态下通常无法进行攻击，除非特殊说明。

---\n
## 2. 基础属性 / Base Stats

### [STAT_HP] 生命值 / Health Points
- **定义**: 角色生存能力的量化指标。
- **归零规则**: 当 HP <= 0 时，角色进入 [死亡] 状态。

### [STAT_MP] 法力值 / Mana Points
- **定义**: 释放技能消耗的资源。
- **恢复**: 每秒恢复 `{{MP_REGEN_BASE}}` 点。

---\n
## 3. 通用交互 / Common Interactions

### [MECH_CRIT] 暴击 / Critical Hit
- **公式**: 最终伤害 = 基础伤害 * `{{CRIT_MULTIPLIER}}` (默认为 1.5)。
- **触发**: 攻击判定时根据 [暴击率] 进行随机判定。

### [MECH_COOLDOWN] 冷却 / Cooldown
- **规则**: 技能释放后进入不可用状态，持续 `{{SKILL_CD}}` 秒。
- **显示**: 界面图标上显示半透明遮罩和倒计时数字。
