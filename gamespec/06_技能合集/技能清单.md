# GameSpec AI 技能清单 (Skill Registry)

> **用途**: 本文件定义了 AI Agent 可调用的所有 specialized skills。
> **指令**: 当用户的需求匹配 [触发条件] 时，Agent **必须 (MUST)** 读取并加载对应的 [技能文件]。

| 技能名称 (Skill Name) | 触发条件 (Trigger) | 技能文件 (Path) |
| :--- | :--- | :--- |
| **Mermaid 绘图专家** | 用户需要生成流程图、状态机、架构图、甘特图时 | `06_技能合集/Mermaid绘图技能.md` |
| **数值模拟专家** | 用户需要设计公式、验证平衡性、计算概率时 | `06_技能合集/数值模拟技能.md` |
| **规则撰写专家** | 用户需要定义具体的交互逻辑、判定条件时 | `06_技能合集/规则撰写技能.md` |
| **逻辑审查专家** | 用户提交创意需要验证、或请求查找漏洞时 | `06_技能合集/逻辑审查技能.md` |
| **文案润色专家** | 涉及文档输出、剧情写作或语言风格调整时 | `06_技能合集/文案润色技能.md` |
| **配置表结构生成专家** | 用户需要定义 Excel 表头、数据类型或 Schema 时 | `06_技能合集/配置表结构生成技能.md` |
| **命名与术语生成专家** | 用户需要为道具、技能、NPC 取名或构建术语表时 | `06_技能合集/命名术语生成技能.md` |
| **竞品拆解与反推专家** | 用户提供竞品参考、需要逆向分析系统设计时 | `06_技能合集/竞品拆解技能.md` |
| **美术需求描述专家** | 用户需要向美术提需求、或生成 Image Gen 提示词时 | `06_技能合集/美术需求描述技能.md` |
